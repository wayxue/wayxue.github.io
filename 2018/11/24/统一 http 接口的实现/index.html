需要注意的问题

get请求时路径上的参数不能出现{}，是一个400 bad request 的错误。
可以通过转义的方式暂时解决，{：%7B，}：%7D。
如：
localhost:8080/ser?method=123&params={}
localhost:8080/ser?method=123&params=%7B%7D
